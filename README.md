# TNKB Package

## What is this?

TNKB stands for *Tanda Nomor Kendaraan Bermotor* (motor vehicle number sign), is a package to validate and decode Indonesian vehicle's plate/registration number.

## Regulations

(*Last updated: Sunday, 24 Oct 2021*)
